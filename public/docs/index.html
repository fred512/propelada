Modos de Acesso

O ProPelada tem dois perfis de acesso com permissões diferentes.

👤 Administrador

  • Acessa com login e senha cadastrados
  • Cria e gerencia partidas
  • Adiciona e edita participantes
  • Configura pontuação e cores dos times
  • Gera códigos de acesso para visitantes
  • Acessa todos os relatórios

👁 Visitante

  • Acessa com código de 6 dígitos fornecido pelo admin
  • Visualiza partidas em andamento
  • Consulta ranking e estatísticas (se o admin liberar)
  • Não pode editar nada
  • Código válido por 3 horas
💡 Para gerar um código de acesso para visitantes, vá até a página Acesso no menu lateral.

1

Criar uma nova partida

Na tela principal da pelada, clique no botão + (criar partida) no canto superior direito. Informe a data e confirme. A partida começa no 1º Tempo.

2

Montar os times

Use o botão + abaixo de cada time para adicionar jogadores da lista de espera. Você pode adicionar vários de uma vez marcando as caixinhas e clicando em Adicionar.

3

Registrar gols e cartões

Toque no nome de um jogador para abrir as ações. Use os botões – e + para gols, e os botões coloridos para cartões. Também é possível registrar gols contra.

4

Iniciar o intervalo

Clique no botão 1º Tempo para abrir o modal de intervalo. Marque quem sai, adicione substitutos e clique em Iniciar 2º Tempo.

5

Encerrar a partida

Ao final, clique no ícone de cadeado no topo para encerrar a partida. Ela ficará salva com todos os registros. Você pode ver o resumo completo pelo botão Resumo (ícone de lista).

Jogadores e Times

Cada partida tem dois times com até 9 jogadores cada.

➕

Adicionar jogador

Clique no botão + abaixo de cada time para escolher jogadores da lista de espera.

↔️

Mover entre times

Abra as ações do jogador e use o botão Mover para Time para transferi-lo para o outro time.

🧤

Goleiro

Goleiros aparecem com o ícone de luva ao lado do nome, facilitando a identificação rápida.

🔴

Remover jogador

Abra as ações do jogador e use o botão vermelho Remover da Partida. O jogador volta à lista de espera.

⚠️ Cada time deve ter exatamente 9 jogadores por tempo. O sistema alerta quando o número está incorreto, especialmente no intervalo.
🎯

Ações do Jogador

Toque no nome de qualquer jogador para registrar ocorrências da partida.

⚽ Gols

  • Use os botões – e + para somar ou subtrair gols
  • Há um contador separado para gols contra
  • Os gols são contabilizados por tempo (1º ou 2º)

🟨 Cartões

  • 🟨 Amarelo — advertência
  • 🟦 Azul — suspensão temporária
  • 🟥 Vermelho — expulsão
  • Cartões acesos ficam com brilho colorido. Clique novamente para remover.

🔄 Substituição

  • Não — jogador ainda em campo (padrão)
  • 1º Tempo — substituído durante o primeiro tempo
  • Intervalo — substituído na pausa entre os tempos
  • Quando substituído, os botões de gol e cartão ficam desabilitados
📋

Lista de Espera

Jogadores que aguardam para entrar na partida ficam na lista de espera.

1

Acessar a lista

Clique no ícone de lista com número (🟢) no topo da tela de partida. O número indica quantos jogadores estão aguardando.

2

Adicionar ao time

Clique no + abaixo de um time, marque os jogadores desejados na lista e toque em Adicionar. É possível selecionar vários de uma vez.

3

Remover da lista

Abra a lista de espera e toque no ícone de lixeira ao lado do jogador que deseja remover.

🔄

Intervalo

O intervalo é o momento de fazer substituições antes de iniciar o 2º Tempo.

1

Abrir o intervalo

Clique no botão 1º Tempo quando o primeiro tempo terminar. O modal de intervalo será exibido.

2

Marcar quem sai

Para cada jogador que vai sair, clique em Sai. O botão muda para Volta caso queira desfazer. O contador de jogadores atualiza em tempo real.

3

Adicionar substitutos

Use o botão + Espera de cada time para trazer jogadores da lista de espera. Eles são marcados automaticamente como entrou no intervalo.

4

Iniciar 2º Tempo

Certifique-se que cada time tem 9 jogadores (contador fica verde). Então clique em Iniciar 2º Tempo.

⚠️ O contador fica laranja se o time tiver mais ou menos de 9 jogadores. Ajuste antes de iniciar o 2º tempo.

1️⃣

Aba 1º Tempo

Jogadores que jogaram o primeiro tempo com seus gols e cartões deste período.

2️⃣

Aba 2º Tempo

Jogadores que jogaram o segundo tempo com seus gols e cartões deste período.

💡 O resumo pode ser acessado a qualquer momento durante a partida, clicando no botão de lista no topo da tela.

Relatórios

Documentos detalhados para conferência e transparência da pontuação.

📊 Tabela de Pontuação

  • Lista completa de todos os jogadores
  • Gols, cartões, presenças e resultados
  • Pontuação total calculada automaticamente
  • Ideal para ver o ranking de pontos

🔍 Relatório de Conferência

  • Detalhamento por categoria de pontuação
  • Presenças, faltas, jogos assistidos
  • Bônus (ex: jogo na chuva)
  • Permite verificar cada ponto atribuído
💡 Os relatórios abrem em uma nova janela para facilitar a impressão ou exportação como PDF pelo navegador.

Participantes

Cadastro e gestão de todos os jogadores da sua pelada.

Campos do Cadastro

  • Nome e Apelido — identificação do jogador
  • Celular e Data de Nascimento — dados de contato
  • Tipo: Jogador, Ex-Jogador, Convidado, Goleiro, Juiz
  • Posição: Zagueiro, Meia, Lateral, Atacante
  • Nível: Bom, Médio, Ruim — classificação de desempenho
  • Faixa etária: Novo, Médio, Velho
  • Velocidade: Rápido, Médio, Lento
  • Foto — imagem de perfil do jogador
💡 Use a barra de busca para encontrar jogadores rapidamente pelo nome ou apelido. Clique em um resultado para carregar os dados no formulário e editar.

Configurações

Personalize a pelada de acordo com as regras do seu grupo.

👁

Exibir Estatísticas

Controla se visitantes podem acessar as estatísticas e ranking da pelada. Ative ou desative com o botão de alternância.

📅

Período dos Gráficos

Define o intervalo de datas padrão para exibição nas estatísticas e gráficos.

🎨

Cores dos Times

Escolha entre Verde, Azul, Vermelho ou Amarelo para cada time. A cor aparece nos cards dos jogadores na tela de partida.

🏅

Critérios de Pontuação

Configure quantos pontos valem: jogo disputado, jogo assistido, vitória, empate, derrota, jogo na chuva e cada tipo de cartão.

Código de Acesso

Gere e compartilhe códigos para que visitantes possam ver a pelada.

1

Gerar novo código

Na página Acesso, clique em Gerar novo código. Uma confirmação será exibida avisando que o código anterior será invalidado.

2

Compartilhar

O código de 6 dígitos é exibido na tela com o horário de expiração. Compartilhe com quem quiser dar acesso de visitante.

3

Validade

Cada código é válido por 3 horas. Após expirar, um novo código deve ser gerado para continuar acessando.

⚠️ Ao gerar um novo código, o código anterior é imediatamente invalidado. Visitantes que estavam usando o código antigo precisarão do novo.
